# Primeiros Passos Com o Java

#### Na primeira aulda aprendi a como compilar e executar uma aplicação java com os comamdo:

- **Javac**: Para Compilar o codigo java, criando um arquivo .class e transformando meu codigo java em um bytecode para ser lide pelo **JVM**.

- **Java**: Para execução da minha aplicação em Java.

Crie um simples programa que imprime a famosa frase da programação: **Olá Mundo!!**.

```bash
public class PrimeirosPassos {
    public static void main(String[] args) {
        System.out.println("Olá mundo!!");
    }
}
```
